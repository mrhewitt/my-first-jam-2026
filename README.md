
# My First Jam 2026 : Worms<sup>3</sup>

Game jam entry for [My First Jam 2026](https://itch.io/jam/my-first-jam-2026) - Worms<sup>3</sup>

See it in action on Itch.io:  [Worms Cubed](https://https://daddio-dragonslayer.itch.io/worms-cubed)

Basic clone of a Cubes 2048 type game. My goal was to experiment with multi-player and 3D.

The game is functional, but multi-player does have a lag/jitter when run properly over remote server, so
it needs to be handled in a different way, but was good experience in jam period.
